# jenkins3d-unity-sample
## 概要
Jenkinsユーザカンファレンス2015のLTで発表したデモのUnityプロジェクトです。
Jenkins 3D Logo をUnity内で表示したり動かしたりしています。

### Jenkins
http://jenkins-ci.org/

### Jenkins Logo
https://wiki.jenkins-ci.org/display/JENKINS/Logo

## ライセンス
オリジナルの3dデータに従います。
LISENCEファイルを参照。
